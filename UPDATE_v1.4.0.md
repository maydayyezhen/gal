### 3. 完整的对话编辑功能

每条对话可以设置：
- ✅ **说话者** - 输入框填写
- ✅ **角色立绘** - 下拉选择或导入
- ✅ **对话内容** - 文本框输入
- ✅ **顺序调整** - ↑↓ 按钮
- ✅ **删除** - 删除按钮

## 🎮 完整工作流程

### 创建一个场景

```
1. 点击"+ 添加新场景"

2. 设置场景基础信息
   场景ID: scene_ch1_classroom
   
3. 选择/导入背景
   方式A: 下拉选择现有背景
   方式B: 点"+ 导入背景"
          → 选择文件 classroom.png
          → 命名 classroom
          → 下载 bg_classroom.png
          → 放到 images/backgrounds/
   
4. 选择/导入BGM
   方式A: 下拉选择现有音乐
   方式B: 点"+ 导入BGM"
          → 选择文件 daily.mp3
          → 命名 daily
          → 下载 bgm_daily.mp3
          → 放到 audios/bgm/

5. 添加对话
   点"+ 添加对话"
   
6. 编辑第1条对话
   说话者: 阳菜
   立绘: char_hina_happy (或点"+ 导入立绘")
   内容: 早上好！
   
7. 添加更多对话
   重复步骤5-6
   
8. 调整对话顺序
   用 ↑↓ 按钮
   
9. 保存场景
```

### 导入资源详细步骤

**导入角色立绘示例：**
```
1. 准备文件
   hina_happy.png (你的图片文件)

2. 在对话编辑卡片中点"+ 导入立绘"

3. 填写导入信息
   选择文件: [浏览] → 选择 hina_happy.png
   资源名称: hina_happy
   
4. 点"确认导入"
   → 自动下载为 char_hina_happy.png
   
5. 弹窗提示
   ┌────────────────────────┐
   │ ✅ 文件已下载:          │
   │ char_hina_happy.png    │
   │                        │
   │ 请将文件放入项目目录:   │
   │ images/characters/     │
   │ char_hina_happy.png    │
   │                        │
   │ 资源ID: char_hina_happy│
   └────────────────────────┘
   
6. 放置文件
   将下载的 char_hina_happy.png
   复制到 game_project/images/characters/
   
7. 自动应用
   该立绘自动选中
   下次编辑其他对话时也能选择
```

## ⚠️ 注意事项

### 资源命名
- 用小写英文
- 用下划线分隔
- 不要用中文
- 不要用空格

**好的命名：**
- school
- hina_happy
- daily
- battle_01

**不好的命名：**
- 学校 ❌
- Hina Happy ❌
- daily-music ❌

### 文件放置
导入资源后：
1. 找到下载的文件
2. 看清楚提示的目录
3. 复制到正确位置
4. 刷新游戏测试

### 对话编辑
- 用 ↑↓ 调整顺序
- 删除前确认
- 每条对话都能独立编辑
- 立绘可选可不选
