## ✨ 新功能详解

### 自动类型识别

**图片文件：**
- `bg_*.png` → 自动识别为背景 → `images/backgrounds/`
- `char_*.png` → 自动识别为角色 → `images/characters/`
- 其他 → 放入 `images/others/`

**音频文件：**
- `bgm_*.mp3` → 背景音乐 → `audios/bgm/`
- `se_*.mp3` → 音效 → `audios/se/`
- `voice_*.mp3` → 语音 → `audios/voices/`
- 其他 → 默认音效 → `audios/se/`

### 下载功能

点击"💾 下载"按钮后：
1. 文件自动下载到浏览器下载目录
2. 弹窗提示具体放置路径
3. 你只需复制粘贴到指定位置

### 命名规则说明

界面内直接显示：
- 详细的命名规则
- 实际示例
- 颜色标注的代码
- 温馨提示

## ⚠️ 注意事项

### 文件下载后的操作

1. **找到下载的文件**
   - Windows: `C:\Users\你的用户名\Downloads\`
   - Mac: `~/Downloads/`

2. **复制到项目目录**
   - 根据提示的路径
   - 例如：`images/backgrounds/bg_school.png`

3. **测试游戏**
   - 双击 `index.html`
   - 检查资源是否正常显示

### 如果章节没显示

1. 刷新浏览器（F5）
2. 查看控制台（F12）是否有错误
3. 确认 `chapters/chapter1.js` 文件存在
