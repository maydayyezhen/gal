## 💡 使用建议

### 资源准备
为了最佳预览效果：
- 背景图：建议 1920x1080
- 角色立绘：建议 800x1200
- 文件格式：PNG（支持透明）

### 命名规范
- 背景：school, rooftop, classroom
- 立绘：hina_happy, sora_normal
- BGM：daily, mystery, battle

### 编辑流程
```
1. 先设置背景和BGM
   （浏览卡片，点击选择）

2. 逐条添加对话
   （左侧选立绘，右侧写内容）

3. 调整顺序
   （用 ↑↓ 按钮）

4. 设置属性
   （位置、动画、音效）

5. 保存场景
```
